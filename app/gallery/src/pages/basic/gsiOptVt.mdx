最適化ベクトルタイルを使った最も簡単なレイヤの生成方法です。

`@cieloazul310/ol-gsi-vt` モジュールから `gsiOptVtLayer` 関数をインポートし OpenLayers レイヤーとして定義するだけで、`VectorTile` レイヤが生成されます。オプションを設定しない場合は、プリセットのスタイルが適用されます。

## Example code

```ts
import { gsiOptVtLayer } from '@cieloazul310/ol-gsi-vt';

const layer = gsiOptVtLayer();

export default layer;
```

export default ({ children }) => <MdxMapLayout meta={meta}>{children}</MdxMapLayout>;
