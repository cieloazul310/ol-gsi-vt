最適化ベクトルタイルを使った淡色レイヤの生成方法です。

`@cieloazul310/ol-gsi-vt` モジュールから `gsiVtPaleLayer` 関数をインポートし OpenLayers レイヤーとして定義します。淡色プリセットのスタイルが適用されたレイヤが生成されます。

## Example code

```ts
import { gsiVtPaleLayer } from '@cieloazul310/ol-gsi-vt';

const layer = gsiVtPaleLayer();

export default layer;
```
