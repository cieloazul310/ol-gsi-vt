ベクトルタイルを使った最も簡単なレイヤの生成方法です。

`@cieloazul310/ol-gsi-vt` モジュールから `gsiVtLayer` 関数をインポートし OpenLayers レイヤーとして定義します。オプションを設定しない場合は、プリセットのスタイルが適用されます。

## Example code

```ts
import { gsiVtLayer } from '@cieloazul310/ol-gsi-vt';

const layer = gsiVtLayer();

export default layer;
```
